    Project Overview

    This system is a smart monitoring dashboard designed for Arowana tanks. It uses YOLOv11 to detect fish behavior such as hunger.

    Objectives

    • Monitor Arowana fish behavior using AI.
    • Detect hunger.
    • Log AI-triggered actions in the activity history.
